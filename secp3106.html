Group Project

E-ducator | Flutter & Supabase

Academic Management Mobile App for IKM Johor Bahru

Project Overview: E-ducator is a Flutter-based mobile application developed to help Institut Kemahiran Malaysia (IKM) Johor Bahru manage their daily academic operations digitally. The app replaces manual, paper-based processes with a centralised digital platform, covering attendance recording, timetable management, room booking, discipline tracking, and reporting.

My Contributions: As part of the development team, I was specifically responsible for building and integrating two core components of the application: Module 1 (Student Attendance) and Module 5 (Timetable View).

Module 1 — Student Attendance

I developed this module to allow lecturers to easily view their assigned classes for the day and seamlessly record student attendance. The interface provides quick toggle buttons for Present, Absent, or Late. I ensured the data syncs in real-time to our Supabase database, allowing administrators to track attendance patterns immediately without manual data entry.

Module 5 — Timetable View

I designed and implemented this module to display a lecturer's personal timetable, showing all their classes grouped by day of the week. Each class slot dynamically displays the subject name, subject code, class group, room, and time. I also added filtering (by day or subject) and keyword search functionalities. To improve user experience, I programmed the class slots to act as navigation buttons—tapping on any class smoothly redirects the user directly to Module 1 to take attendance for that specific session.
